# Prashant Maurya Portfolio

A modern, responsive portfolio website for Prashant Maurya, a Backend & AI Developer.

## Features
- Clean, minimal design with vibrant colors
- Fully responsive across all devices
- Smooth animations and interactions
- Skills, projects, and contact sections
- Direct links to Fiverr, GitHub, LinkedIn

## Technologies
- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts

## Deployment

### Option 1: Vercel (Recommended)
1. Push this repository to GitHub.
2. Connect your GitHub repo to Vercel.
3. Deploy automatically – no build step needed.

### Option 2: Netlify
1. Push to GitHub.
2. Connect to Netlify.
3. Deploy as static site.

### Option 3: GitHub Pages
1. Enable GitHub Pages in repository settings.
2. Set source to main branch.
3. Site will be available at `https://yourusername.github.io/repository-name/`

### Option 4: Manual Upload
Upload the files (`index.html`, `style.css`, `script.js`) to any static hosting provider.

## Local Development
Run `python -m http.server 8000` in the project directory and visit `http://localhost:8000`.
